## §2 小夢選單(三層導流)

### 2.1 三層導流架構

```
第一層:LINE 對話(接收祝福 / 接收通知)
   ↓
第二層:小夢靈性導引(引導式對話 / 解讀報告)
   ↓
第三層:小夢神殿 網站(深度互動 / 占卜儀式)
```

### 2.2 每層定位

| 層級 | 場景 | 角色 | 裝置 |
|---|---|---|---|
| **第一層** | 接收推播祝福 | 通知匣 | LINE App |
| **第二層** | 對話式引導 | 小夢靈性導引 | LINE App |
| **第三層** | 深度互動儀式 | 小夢神殿 | LIFF Web |

### 2.3 三層導流實作

#### 第一層觸發:
- 加入好友
- 每日推播
- 節氣推播
- 抽牌完成通知
- 靈性積分入帳通知

#### 第二層觸發:
- 使用者輸入「開始」、「占卜」、「我要算」任一關鍵字

- 小夢靈性導引主動詢問使用者問題

#### 第三層觸發:
- 第二層對話結束後提供 LIFF URL
- 點選「立即前往小夢神殿」按鈕
- 自動帶入 session token


## §5 會員綁定

### 5.1 綁定流程

```
使用者加入 LINE 好友
   ↓
LINE userId 自動產生
   ↓
使用者首次開啟 LIFF(從 Rich Menu)
   ↓
LIFF SDK 取得 userId + accessToken
   ↓
POST /api/line/bind { userId, accessToken }
   ↓
伺服器檢查 LINE profile → 取得 displayName + pictureUrl
   ↓
建立/更新用戶資料:
   {
     lineUserId,
     displayName,
     pictureUrl,
     bindAt: ISO timestamp,
     points: 50  ← 首次綁定獎勵
   }
   ↓
localStorage: { lineBound: true, points: 50 }
   ↓
第一層推播:
「✨ 你已成功加入小夢神殿 ✨
50 靈性積分已入帳,願星光指引你。」
```

### 5.2 綁定後資料儲存

| 欄位 | 來源 | 用途 |
|---|---|---|
| `lineUserId` | LINE | 唯一識別 |
| `displayName` | LINE profile | 顯示名 |
| `pictureUrl` | LINE profile | 頭像 |
| `bindAt` | 系統 | 綁定時間 |
| `points` | 系統 | 靈性積分 |
| `tier` | 系統 | 會員階級 |
| `firstFreeUsed` | 系統 | 首抽免費是否已用 |
| `lastShareAt` | 系統 | 上次分享時間 |
| `shareCount` | 系統 | 今日分享次數 |

### 5.3 解綁規範

使用者點 Rich Menu 「加入小夢」可重新觸發綁定流程,但**不得刪除歷史占卜紀錄**(隱私保護)。

## §9 登入流程

### 9.1 三種登入方式

| 方式 | 適用 | 實作 |
|---|---|---|
| **LINE LIFF 登入** | LINE App 內 | liff.login() + accessToken |
| **網頁訪客登入** | 桌面 / 行動瀏覽器 | localStorage uuid |
| **管理員後台** | `admin.html` | 密碼 `tarot2026`(已對齊 line-setup.md) |

### 9.2 LINE LIFF 登入流程

```
使用者點 Rich Menu 任一入口
   ↓
LIFF SDK init (liff.init)
   ↓
檢查 liff.isLoggedIn()
   ↓ (false)
liff.login() → LINE OAuth
   ↓ (成功)
取得 idToken + accessToken + profile
   ↓
POST /api/liff/login { idToken, profile }
   ↓
伺服器驗證 idToken(用 LINE_CHANNEL_SECRET)
   ↓
建立 session token(JWT)
   ↓
回傳 session token → 前端 localStorage
   ↓
頁面載入用戶狀態(姓名、頭像、積分)
```

### 9.3 網頁訪客登入流程

```
使用者首次開啟網站(非 LINE 內)
   ↓
檢查 localStorage.uuid
   ↓ (不存在)
生成 uuid → 儲存 localStorage
   ↓
建立訪客 session(無 LINE 綁定)
   ↓
給予 50 靈性積分(僅首次)
   ↓
顯示「✨ 訪客模式 — 加入 LINE 好友獲得更多祝福 ✨」
```

### 9.4 管理員登入流程

```
瀏覽器開啟 /admin.html?admin=1
   ↓
彈出密碼輸入框
   ↓
驗證密碼(目前為 tarot2026)
   ↓ (成功)
sessionStorage.setItem('adminSession', token)
   ↓
顯示後台介面
   ↓
右上角顯示「🌙 連回前台」按鈕(已實作)
```

### 9.5 Session 管理

| 層級 | 儲存 | 期限 |
|---|---|---|
| LINE accessToken | LIFF SDK | LIFF lifetime |
| session token (JWT) | localStorage | 30 天 |
| admin session | sessionStorage | 8 小時(瀏覽器關閉即失效) |

### 9.6 登入失敗處理

#### 401 Unauthorized
```js
window.location.href = '/liff-login.html?return=' + encodeURIComponent(window.location.pathname);
```
