## 3 滿 599 折抵 50 積分(老闆硬規則)

### 3.1 計算公式

```
購買 $599 完整牌陣:
  $599 - (50 積分 × $1) = $549 實付
  ↓
綠界付款 $549
  ↓
points_log: -50(兌換)
orders: 寫入 $549
```

### 3.2 規則

- 1 張訂單只能折抵 1 次
- 折抵後不退 50 積分(已用掉)
- 退款時 50 積分退回(全額退款)

## 4 綠界 ECPay 串接流程

```
使用者 → 選商品 → 結帳頁
  ↓
綠界付款頁(3D 驗證)
  ↓
綠界 webhook
  ↓
/api/ecpay/return
  ├─ 1. 驗證 CheckMacValue
  ├─ 2. 寫入 orders
  ├─ 3. 發 points_log(+)
  ├─ 4. 升級 tier(若訂閱)
  ├─ 5. 推播 LINE 通知
  └─ 6. 寄信(若買實體)
```
