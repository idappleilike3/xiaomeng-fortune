═══════════════════════════════════════════════════════════

【重要】Official Account Manager「歡迎訊息」通常只能放純文字，
無法放 Flex / postback 按鈕（開始解碼、查看所有方案）。
這不是 bug，是 Console 能力限制；也無法用 API 把 Flex 寫進 Console 問候語。

【真正帶按鈕的歡迎】= Messaging API 的 follow 事件
server 收到加好友後，自動 reply Flex（含按鈕）到對方聊天室：
  [1] 開始解碼     → postback  funnel=start
  [2] 查看所有方案 → URI      LIFF /pricing 或
      https://xiaomeng-fortune.onrender.com/erosee-l2-pricing.html

Webhook 必須是：
  https://xiaomeng-fortune.onrender.com/api/line/webhook
且在 LINE Developers 已啟用，follow 才會觸發。

測試：取消追蹤 OA → 再加回好友，聊天室應出現帶按鈕的 Flex。
