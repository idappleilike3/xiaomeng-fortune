## 正式素材建議

- 格式：`mp4`（H.264）+ 可選 `webm`
- 直式約 `540×900` 或 `720×1280`
- 長度：3–8 秒可無縫循環
- 必須可靜音播放（`muted` + `loop` + `playsinline`）
- 內容：牌面微動、光暈、煙霧、緩慢翻轉皆可

把新檔案用**同名**覆蓋此資料夾即可，不用改程式。
