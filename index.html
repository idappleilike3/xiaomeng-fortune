安心說明

為什麼小夢老師需要這些資料？

  • 出生日期用來計算生命靈數、流年與基本命盤。
  • 出生時間用來細分八字時辰與紫微斗數命宮；不知道也可以先略過。
  • 出生地用來校正時區與地點，讓命盤更貼近你的出生條件。
  • 暱稱與性別讓解析口吻更自然，也可用於合盤與關係分析。

資料只用於命理解析與個人化提醒。正式上線後會提供刪除資料與取消提醒的入口。

LINE Setup

之後真正串 LINE 需要這些東西

  1. LINE 官方帳號建立品牌帳號，設定歡迎訊息與圖文選單。
  2. LINE Developers啟用 Messaging API，取得 Channel secret 與 Channel access token。
  3. Webhook URL部署後填入 https://你的網址/api/line/webhook，讓 LINE 把訊息送到伺服器。
  4. LIFF App把這個網頁網址放進 LIFF，使用者可在 LINE 內開啟前台工具。
  5. Node Webhook目前已建立 server.js，可回覆塔羅、求籤、生命靈數、命盤、MBTI 與市集指令。
  6. 資料庫用 Supabase 儲存會員、占卜紀錄、商品點擊與訂單狀態。
