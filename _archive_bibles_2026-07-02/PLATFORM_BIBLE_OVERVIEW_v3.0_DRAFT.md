### 第四幕:系統主分類
- 命運塔羅 → ❤️ 感情 / 💼 工作 / 💰 財富 / 🌙 自我探索 / ✨ 更多牌陣
- (毛孩 / 神諭 同樣 5 個 L2 入口,對齊老闆指示)

- (其他主題同結構)

### 第六幕:牌陣(對齊老闆 7:00)
- 範例:三張牌 / ★★★★★ / 約 5 分鐘 / 今日免費 / 開始探索

### 第七幕:第一次才登入(對齊老闆 7:00 重要原則)
- **這一步才跳登入**
- 文案:「加入神殿」(**不是**「登入」)
- 4 個價值鉤子:
  - ✔ 保存命運紀錄
  - ✔ 開啟今日免費
  - ✔ 領取 50 SP
  - ✔ 累積探索等級
- 2 顆按鈕:立即加入 / 稍後再說

### 第八幕:成功(對齊老闆 7:00)
- 「歡迎來到神殿」
- 🎁 50 SP 已入帳
- 🎁 命運塔羅 首次免費
- 🎁 毛孩心語 首次免費
- 🎁 今日神諭 首次免費
- 「是不是。很有儀式。」(老闆原文)

### 第九幕:開始探索
- 閉上眼睛 → 默念你的問題
- 洗牌 → 切牌 → 展牌 → 選牌 → 翻牌 → 深度解讀

### 第十幕:完成
- 「今晚的探索完成」
- 獲得 +5 SP
- 已收藏
- 分享 LINE / FB / 返回首頁
