## ⑤ 重新規劃靈性積分(Soul Points)

### 5.1 積分不只是數字 — 完整系統

| 維度 | 規格 |
|---|---|
| **目前積分** | localStorage + Server 同步 |
| **積分來源** | 見 5.2 |
| **積分使用** | 見 5.3 |
| **積分紀錄** | 不可變 ledger(server side) |
| **有效期限** | 取得後 365 日未用歸 0 |
| **積分失效** | 每月 1 號 cron job |
| **購買積分** | $100 = 100 SP / $500 = 550 SP(+50) / $1,000 = 1,150 SP(+150) |
| **交易紀錄** | 完整 audit log(誰、何時、多少、為何) |

### 5.2 積分 4 大來源

| 來源 | 數量 | 限制 |
|---|---|---|
| 註冊禮 | +50 | 1 次 |
| 每日簽到 | +5 | 1 次/日 |
| 占卜完成 | +2 | 每次 神聖探索儀式 |
| 分享好友 | +50 | 1 次/日(防洗) |
| 推薦付費 | +200 | 被推薦者首次付費 |
| 訂閱月禮 | +50(尊榮)/ +200(殿堂) | 每月 |
| 連續 7 日 | +100 | 7 日連續登入 |
| 生日禮 | +500 | 生日當月 |

### 5.3 積分 4 大去處

| 去處 | 價格 | 限制 |
|---|---|---|
| 兌換完整牌陣 | -50 | 1 次/日 |
| 折抵 599 方案 | -1,000 | 滿 599 可折抵 50 積分(老闆硬規則) |
| 抽獎 | -200/次 | 週抽 1 次 |
| 商城兌換 | 浮動 | Plus 以上 |

**⚠️ 「滿 599 可折抵 50 積分」是老闆 6:55 硬規則,對齊 §7 商城流程。**

### 5.4 不變原則

```
- 任何 +/- 必須留 log(誰、何時、多少、為何)
- 不可逆(不能直接改 balance,只能加 -X 的紀錄)
- 過期:取得後 365 日未用 → 歸 0
- 反作弊:同 IP / 裝置指紋 → 拒絕
```

```

  ↓

  ↓

  ↓

  ├─ 2. 寫入 orders
  ├─ 3. 發 points_log(+)
  ├─ 4. 升級 tier(若訂閱)
```
