## 1. User Journey(使用者旅程)

### 1.1 5 階段旅程圖

```
陌生訪客
  ↓ 點 LINE / Google / 直接網址
訪客(未登入)
  ↓ 看到 4 入口(L1)→ 進 L2 → 進 L3 → 點牌陣
未登入體驗者
  ↓ 想看完整解讀 / 想存紀錄
新會員(註冊)
  ↓ 用滿 1 次 / 想用更多
活躍會員
  ↓ 升級 / 訂閱 / 解鎖完整牌陣
付費會員
  ↓ 拉朋友 / 創作內容
推廣者(Referral)
  ↓ 終身身份
品牌大使
```

### 1.2 5 階段關鍵指標

| 階段 | 核心 KPI | 對應漏斗 |
|---|---|---|
| 陌生訪客 → 訪客 | 點擊率 / 跳出率 | 流量 |
| 訪客 → 未登入體驗 | 完成 1 次占卜 | 啟動 |
| 未登 → 新會員 | 註冊轉換率 | 註冊 |
| 新會員 → 活躍 | 7 日回訪率 | 啟動 |
| 活躍 → 付費 | ARPU / 付費轉換 | 變現 |
| 付費 → 推廣者 | Referral 觸發數 | 裂變 |

### 1.3 各階段的 UX Hook

| 階段 | Hook | 對齊規範 |
|---|---|---|
| 陌生訪客 | LINE 歡迎詞 + Rich Menu 6 宮格 | LINE spec v2 |
| 訪客 | Hero 4 入口 + L1 「開始探索」 | UX Spec v1.0 |

| 活躍 | 紀錄收藏 + 連續 7 日獎勵 | §11 收藏系統 |

```
┌──────────────┐     ┌──────────────┐

       │                    │
       │ 1:N                │ 1:N
       ↓                    ↓
┌──────────────┐     ┌──────────────┐
│  points_log  │     │  favorites   │

       │                    
       │ 1:N                
       ↓                    
┌──────────────┐     ┌──────────────┐
│  orders      │     │  subscriptions│
│ (訂單)       │     │ (訂閱)       │
└──────────────┘     └──────────────┘
       │                    │
       └────────┬───────────┘
                ↓
       ┌──────────────┐
       │  payments    │
       │ (綠界金流)   │
       └──────────────┘

┌──────────────┐

### 4.2 members 主表欄位(預估)

| 欄位 | 型態 | 說明 |

```
