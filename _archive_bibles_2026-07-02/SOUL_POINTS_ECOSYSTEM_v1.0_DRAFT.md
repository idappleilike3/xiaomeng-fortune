## 3 探索等級 — SP + EXP 雙系統(對齊老闆 7:57)

### 3.1 SP vs EXP 差別

| 維度 | SP | EXP |
|---|---|---|
| 用法 | 換獎勵 | 升等級 |
| 來源 | 付費 / 任務 / 簽到 | 探索行為 |
| 累積 | 會過期(365 日) | 永久(除非重置) |
| 顯示 | 餘額 | 等級 + 進度條 |

| 推薦付費 | +100 EXP |
