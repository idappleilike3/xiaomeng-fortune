## §1 品牌定位 (Brand Positioning)

### 1.1 核心品牌
**小夢老師** — 靈性世界的陪伴者

### 1.2 一句話定位
> 「我不預言未來,而是陪伴每一位使用者,看見屬於自己的答案。」

### 1.3 品牌屬性
| 屬性 | 強度 | 說明 |
|---|---|---|
| 神秘 | 10/10 | 深邃、不可測 |
| 高級 | 10/10 | 香檳金、絲綢質感 |
| 療癒 | 9/10 | 溫柔、撫慰 |
| 精品 | 9/10 | 精雕細琢 |
| 儀式感 | 10/10 | 每一次點擊都是儀式 |
| 電影感 | 8/10 | 史詩級視覺 |

### 1.4 品牌人格 (Brand Persona)
- **性別**:女性
- **年齡**:永遠 30 歲
- **語氣**:溫柔但堅定、神秘但不疏離
- **說話方式**:
  - 「親愛的,讓我們一起看看今天的星軌如何指引你」
  - 「星軌正在為你緩緩揭曉」
  - 「命運之門已為你開啟」
- **絕對不說的**:
  - 「AI 為您分析結果」
  - 「系統已完成運算」
  - 「請選擇您的偏好」(改為「請選擇今天想開啟的命運之門」)

```
    ↓
```

## §8 按鈕規範 (Button)

### 8.1 五種變體(全站統一,不得自訂)

#### A. `.btn--primary` 主要行動
```css
background: linear-gradient(135deg, #f5d38b 0%, #c9b88a 100%);
color: #19142e;
border: 1px solid #f5d38b;
padding: 14px 28px;
border-radius: 12px;
font-family: var(--font-brand);
font-weight: 600;
box-shadow: 0 4px 18px rgba(245, 211, 139, 0.35);
transition: all 280ms ease;
```
**Hover**:
```css
transform: translateY(-2px);
box-shadow: 0 8px 24px rgba(245, 211, 139, 0.55);
```
**範例**:`啟動星軌解碼` / `開始儀式` / `加入小夢`

#### B. `.btn--secondary` 次要行動
```css
background: transparent;
color: #f5d38b;
border: 1px solid #f5d38b;
padding: 14px 28px;
border-radius: 12px;
```
**Hover**:
```css
background: rgba(245, 211, 139, 0.08);
```
**範例**:`再抽一張` / `查看解讀`

#### C. `.btn--ghost` 文字按鈕
```css
background: transparent;
color: #c9b88a;
border: 0;
text-decoration: underline;
text-underline-offset: 4px;
```
**Hover**:`color: #f5d38b`

#### D. `.btn--danger` 危險
```css
background: linear-gradient(135deg, #c44569 0%, #8b2848 100%);

border: 1px solid rgba(196, 69, 105, 0.55);
box-shadow: 0 0 24px rgba(196, 69, 105, 0.35);
```

**範例**:`離開儀式` / `清除資料`

#### E. `.btn--locked` 鎖定(付費解鎖)

## §11 光暈 (Glow)

### 11.1 光暈類型

| 類型 | 規格 | 用途 |
|---|---|---|
| **金色光暈 (divine)** | `box-shadow: 0 0 36px rgba(245,211,139,0.55)` | 神聖元素、品牌字、鎖定 |
| **紫色光暈 (mystic)** | `0 0 28px rgba(183,128,255,0.48)` | 神女、神秘感 |
| **粉色光暈 (gentle)** | `0 0 24px rgba(245,184,216,0.4)` | 寵物系統 |
| **白色光暈 (ethereal)** | `0 0 32px rgba(255,255,255,0.6)` | 神諭系統、神聖瞬間 |
| **玫瑰光暈 (warning)** | `0 0 20px rgba(196,69,105,0.45)` | 警示 |

### 11.2 光暈動畫

#### 呼吸光暈 (`divine-glow-pulse`)
```css
@keyframes divine-glow-pulse {
  0%, 100% { box-shadow: 0 0 28px rgba(245,211,139,0.35); }
  50%      { box-shadow: 0 0 48px rgba(245,211,139,0.65); }
}
animation: divine-glow-pulse 1.8s ease-in-out infinite;
```

#### 緩慢光暈 (`gentle-glow`)
```css
@keyframes gentle-glow {
  0%, 100% { box-shadow: 0 0 20px rgba(245,211,139,0.25); }
  50%      { box-shadow: 0 0 32px rgba(245,211,139,0.45); }
}
animation: gentle-glow 3s ease-in-out infinite;
```

### 11.3 光暈使用守則

- ✅ 主要按鈕:divine-glow-pulse
- ✅ 鎖定元素:divine-glow-pulse
- ✅ 環境元素:gentle-glow
- ✅ 新訊息紅點:divine-glow-pulse(1.5s)
- ❌ 不得在內文段落加光暈(會影響閱讀)
- ❌ 不得在同一畫面超過 5 個元素同時呼吸光暈(會太刺眼)

❌ 商業炫技動畫(3D 翻轉、文字亂碼)
